# Security Policy

## Supported Versions

Security fixes are applied on `main`.

| Version | Supported |
| --- | --- |
| `main` | yes |

## Reporting a Vulnerability

Please report vulnerabilities privately using GitHub Security Advisories for this repository.

If that is not available, open a private channel with the maintainer and do not file a public issue with exploit details.

For reports, include:

- affected commit/tag and environment details
- clear reproduction steps and expected/actual behavior
- impact assessment and any proof-of-concept artifacts

You can expect an initial acknowledgment within 7 days.
